取引所

取引所の役割

取引所は、株・為替・仮想通貨などの「売りたい人」と「買いたい人」を自動的にマッチングさせる仕組みを提供する。 公正な価格形成、流動性の確保、取引の透明性維持などが主な役割。 将来的にAIボットがこの構造をどのように利用できるかを探っていく。

主要な取引所

  • 🇺🇸 NYSE / NASDAQ – 世界最大の株式市場。流動性と情報量が圧倒的。
  • 🇯🇵 東京証券取引所（TSE） – 日本の中心取引所。APIは制限が多い。
  • 🇪🇺 Euronext / Xetra – 欧州各国の主要市場。ETF取引が活発。
  • ₿ Binance / Bybit – 仮想通貨市場。APIが豊富でボット構築に最適。
  • 💱 OANDA / FXCM – 為替取引プラットフォーム。C#やPython対応APIあり。

どの様に分析するのか

各取引所から取得できるデータ（価格、出来高、板情報、ボラティリティ、ニュース）を統合し、 市場ごとの「健康度（MarketHealthScore）」を算出する。 このスコアを基に、ボットがどの市場に資金を振り分けるか判断する。

取引所の情報を受け取れるAPI

  • Alpha Vantage – 無料で株価や為替データを取得できるが、売買は不可。
  • Alpaca – 米国株の自動売買が可能。Paper Trading対応。
  • Binance API – 仮想通貨の取引・価格取得。Bot向けに最適。
  • OANDA API – 為替取引用API。C#から利用しやすい。
  • Interactive Brokers (IBKR) – 世界中の株・ETF・FXに対応する強力API。
